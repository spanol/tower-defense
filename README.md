# Multiplayer Tower Defense

Real-time cooperative/competitive tower defense game running in the browser.

## Setup

```bash
pnpm install
pnpm dev        # Start client dev server
```

## Project Structure

- `client/` — Phaser 3 game client (TypeScript + Vite)
- `server/` — Game server (Phase 3, placeholder)
- `shared/` — Shared types, tower/enemy configs, map data

## Current Status

Phase 1: Core engine + single-player prototype.
